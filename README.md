## 브랜치 변경하기

- 브랜치란 : 기능 변경을 하고 싶을 때 생성 및 사용. 즉, 기존 내용을 유지한 채 새로운 내용을 추가하고 싶을 때 사용한다.
- checkout : 특정 브랜치(혹은 커밋) 으로 돌아가고 싶을 때 사용.
- 소스트리의 체크아웃 : 브랜치 이름을 더블 클릭하는 것만으로도 체크아웃 가능
